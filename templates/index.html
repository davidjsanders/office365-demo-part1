Connecting to Office 365 Services

Office 365 is a public Software as a Service subscription cloud offering from Microsoft. It is designed to provide modern productivity, collaboration, and communication for teams big and small.

Page is {{ method }} mode.

There are a number of parts to this exercise and you do need to make sure these are performed before clicking on the link to "Login...". There's guidance for each step (and references and source links, where appropriate) for each topic.

  1. Create (or use) a demonstration or development Office 365 (O365) tenant
  2. Access the Azure Active Directory for your O365 tenant
  3. Register the app in Azure AD
  4. Get (and copy) the App ID into a configuration file
  5. Grant permissions for the application
  6. Run the app
  7. Generate the KeyCredentials section for the app manifest
  8. Copy the KeyCredentials section into the app manifest in Azure AD
  9. Create a certificate and a key (More info)

You can try it for yourself using this demo: Login to Office 365 with an administrative logon
